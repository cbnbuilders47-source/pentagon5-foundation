# Security Policy

## Supported versions

Milestone 1 contains repository structure and configuration only. There is no released or supported application version.

## Reporting a vulnerability

Do not disclose suspected vulnerabilities in a public issue. Report them through the project's approved private security channel and include a concise description, reproduction details, impact, and any suggested remediation.

Do not include credentials, personal data, or production data in a report. Maintainers will acknowledge the report and coordinate next steps through the same private channel.
